# Learning Guide Appendices

This folder contains supplemental, date-stamped notes that should not duplicate
the canonical guide at the repository root:

- [`../LEARNING_GUIDE.md`](../LEARNING_GUIDE.md) is the main reference for AI
  engineering design principles and best practices.
- [`OFFICE_HOURS/`](OFFICE_HOURS/) contains dated addenda from office-hours
  discussions and similar conversations.

Use this folder for material that has a clear source date or context, such as
office-hours lessons, project-specific retrospectives, or decision records. Keep
general principles, reusable checklists, and broad best-practice guidance in the
root `LEARNING_GUIDE.md` file.
